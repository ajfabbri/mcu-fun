# PNG to C Array Bitmap

Convert an image to C code that defines a bitmap array.

Created to convert a .png image file to a monochrome (1-bit) C array definition
that can be pasted into arduino sketches and used to paint an OLED display.

## Quick Start

1. Create a virtual environment (optional but recommended), and activate it:

```
python -m venv venv
source venv/bin/activate

```

2. Install the package:

```
python -m pip install png2carray
```

3. Run the script:

```
png2c my_image.h
```
